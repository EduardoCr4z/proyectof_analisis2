# Frontend Universidad

React con Keycloak para los roles `Estudiantes`, `Profesores` y `Admin`.

## Configuracion Keycloak

En el cliente `front_react` agrega:

- Valid redirect URIs: `http://localhost:3000/*`
- Web origins: `http://localhost:3000`

## Comandos

```bash
npm install
npm run dev
```

El frontend queda en `http://localhost:3000`.
